# Add Markdown and JSON Reporters

## Summary

Milestone 9 replaces ad hoc CLI printing with explicit Markdown and JSON reporters.

## Goals

- Produce a readable Markdown report with summary, naming conventions, and grouped findings.
- Produce valid JSON for automation consumers.
- Include severity counts in report summaries.
- Keep CLI orchestration thin.

## Non-goals

- No PR comment posting.
- No configuration file.
- No custom theming.
